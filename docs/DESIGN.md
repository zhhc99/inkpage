# inkpage 设计原则

## 项目定位

inkpage 是一个轻量, 随处可用的 web 画布应用. 可用手指或手写笔书写流畅的墨迹.

单文件架构, 不依赖构建工具, 不依赖外部 JS 库, 一个 HTML 文件即可在浏览器中运行. 在各种设备 (桌面/平板/手机, 鼠标/触控/手写笔) 上提供一致的书写体验. 用户可以将其用作临时白板, 也可以通过保存/加载文件来持久化管理墨迹笔记.

## 核心原则

1. **简洁**: 代码和文档力求精简, 不添加非必要功能. 每行代码都应有存在的理由.
2. **单文件**: 所有 HTML, CSS, JS 集中在一个 `.html` 文件中. 不拆分模块, 不使用框架.
3. **跨设备**: 同时支持鼠标, 触控和手写笔输入. 无压感设备通过速度模拟压感.
4. **流畅**: 通过事件合并, 双画布缓存, 视口裁剪等手段确保书写帧率稳定.
5. **分辨率无关**: 矢量管线生成 Path2D 轮廓, 缩放后不模糊.
6. **无限画布**: 笔画坐标存储在无界文档空间中, 用户可以自由平移和缩放. 导出时自动计算内容边界裁切.

## Material Design 3

inkpage 使用 Material Design 3 (MD3) 设计语言构建界面, 遵循其色彩体系与组件规范.

### 色彩体系

所有界面颜色通过 CSS 自定义属性定义, 构成完整的 MD3 色彩角色映射:

- **主色 (Primary)**: `--md-primary` #6750A4, 紫色调, 用于激活态按钮与滑块控件.
- **表面色 (Surface)**: 三级表面色 (lowest/low/container/high/highest) 形成层次分明的背景体系. 画布区域使用 `--md-surface-container-lowest` (纯白), 工具栏使用 `--md-surface-container`.
- **内容色 (On-*)**: 各表面色对应的文字与图标颜色, 确保无障碍对比度.
- **轮廓色 (Outline)**: 用于分割线, 边框, 提示文字等辅助元素.

MD3 色彩 tokens 通过 `:root` 集中管理, 修改主题只需替换 tokens 值.

### 组件样式

- **工具按钮**: 圆形图标按钮 (FAB 形态), 使用 `--md-shape-corner-full` (9999px) 实现全圆角. 悬浮与按下态使用半透明黑色叠加, 激活态使用 `secondary-container` 背景. 按钮点击时触发自定义 ripple 效果 (CSS transition 实现, 非 JS 动画库).
- **弹出面板**: 颜色选择器, 墨迹微调面板, 项目菜单均使用 MD3 的 Popup Card 模式 -- 固定定位, `--md-shape-corner-l` (16px) 圆角, 双层阴影, 缩放+淡入动画 (`popupIn`).
- **Toast 提示**: 底部居中胶囊形, 深色背景浅色文字, `--md-shape-corner-full` 圆角, 1.8 秒自动消失.

## 适应性布局

inkpage 根据屏幕方向自动切换布局模式, 使用 CSS `@media(orientation:*)` 实现, 无需 JS 参与.

### 横屏模式 (landscape)

- 应用采用水平排列: 左侧工具栏 + 右侧画布.
- 工具栏垂直排列按钮, 宽度固定 72px, 占满全高, 右侧 1px 边框分隔.
- 弹出面板锚定在工具按钮右侧展开, 空间不足时翻转到左侧.

### 竖屏模式 (portrait)

- 应用采用垂直排列: 顶部工具栏 + 下方画布.
- 工具栏水平排列按钮, 高度固定 52px, 占满全宽, 底部 1px 边框分隔.
- 弹出面板锚定在工具按钮下方展开, 空间不足时翻转到上方.

### 自适应策略

- 工具栏与分隔线的尺寸/方向通过 media query 切换, 画布始终填充剩余空间 (`flex:1`).
- 弹出面板的定位逻辑 (`positionPopup`) 在 JS 中根据当前方向计算锚点, 窗口 resize 时自动修正位置.
- 缩放手势: 触控设备支持双指捏合缩放, 桌面设备支持滚轮缩放, 均围绕手指/光标位置进行.

## 工具栏

工具栏按功能分为四组, 通过分隔线区隔:

1. **绘制工具**: 画笔, 橡皮
2. **颜色**: 调色板 (点击弹出莫兰迪色系选择器, 每行 4 色, 含自定义颜色)
3. **模式开关**: 触摸墨迹, 模拟压感, 统一缩放
4. **操作**: 重置画布, 项目 (读取/保存/导出), 墨迹微调 (笔触参数)

**统一缩放** 是一个锁定开关. 开启后缩放固定为 100%, 禁止滚轮和捏合缩放, 但平移仍然可用.

### 调色板

默认提供 6 种莫兰迪色系颜色, 分别对应黑, 蓝, 红, 橙, 绿, 紫的色彩语义. 每行 4 个, 排列为 2 行. 莫兰迪色系没有对应黑色的色值, 因此黑色使用纯黑 (#1D1B20). 所有颜色均在白色背景上具有良好的对比度.

## 本地化

inkpage 支持中文 (zh-CN) 和英文 (en) 双语. 英文为 fallback 语言.

- 语言检测基于 `navigator.language`, 匹配 `zh` 前缀即使用中文, 其余使用英文.
- 所有面向用户的文本 (按钮标题, 提示信息, Toast, 面板标签) 均通过 `t(key)` 函数获取, 不在代码中硬编码字符串.
- 翻译表以扁平键值对存储, 按功能域命名 (如 `tool.pen`, `toast.erased`).
- 动态构建的 UI 组件 (墨迹微调面板, 颜色选择器, 项目菜单) 在每次 `build` 时读取当前语言.

## 文档规范

- 所有文档使用中文汉字书写, 标点符号使用英文标点.
- 文档保持简洁, 系统化, 不冗余.

## 代码规范

- 代码应尽可能简洁.
- 尽量不使用注释, 让代码自述功能. 如果需要注释, 使用英文注释.
